# Visual Design Skill

Esta skill orienta a criação de peças visuais equilibradas, atraentes e funcionais, aplicando princípios fundamentais de design.

## Princípios Fundamentais

1. **Hierarquia Visual**
   - O olhar do usuário deve ser guiado naturalmente pelo layout.
   - Elementos mais importantes devem ter maior peso (tamanho, cor, contraste).
   - Use tamanhos de fonte em proporções harmônicas (ex: escala modular).

2. **Contraste e Acessibilidade**
   - Garanta que o texto seja legível contra o fundo.
   - Use cores contrastantes para botões e chamadas para ação (CTAs).
   - Teste sempre a taxa de contraste (mínimo 4.5:1 para texto normal, 3:1 para texto grande - WCAG AA).

3. **Alinhamento e Grades (Grids)**
   - Elementos devem estar alinhados entre si para criar ordem.
   - Use um sistema de grid (ex: 8pt grid, 12 colunas) para manter a consistência.
   - O alinhamento à esquerda é geralmente mais fácil de ler para blocos de texto longos.

4. **Espaçamento (White Space)**
   - O espaço em branco (ou espaço negativo) não é espaço vazio; é um elemento de design.
   - Agrupe elementos relacionados (Lei da Proximidade).
   - Dê respiro aos elementos para evitar sobrecarga cognitiva.

## Tipografia

- Limite-se a no máximo 2 ou 3 famílias tipográficas por projeto.
- Uma fonte para títulos (pode ser mais expressiva) e uma para corpo de texto (foco na legibilidade).
- Ajuste o espaçamento entre linhas (line-height) para 1.4 a 1.6 vezes o tamanho da fonte para blocos de texto.

## Cores

- **Regra 60-30-10**:
  - 60% Cor Dominante (fundo).
  - 30% Cor Secundária (elementos de suporte).
  - 10% Cor de Destaque (CTAs, alertas).
- Entenda a psicologia das cores (ex: Azul = Confiança, Vermelho = Urgência/Erro, Verde = Sucesso/Crescimento).

## Instruções de Uso

1. **Defina o propósito**: O que o design precisa comunicar?
2. **Estabeleça a grade**: Crie a estrutura antes de adicionar conteúdo.
3. **Aplique o conteúdo bruto**: Coloque textos e imagens sem estilização.
4. **Estabeleça a hierarquia**: Aumente títulos, destaque palavras-chave.
5. **Ajuste o espaçamento**: Refine o respiro entre os elementos.
6. **Aplique cores e tipografia**: Finalize o visual com a identidade da marca.
7. **Teste a acessibilidade**: Verifique contrastes e legibilidade.
