# Skills Collection — Copy, Design & UI/UX

Repositório de skills especializadas para Manus AI, cobrindo as principais disciplinas de **Copywriting**, **Design Visual** e **UI/UX**.

## Estrutura do Repositório

```
skills-collection/
├── copywriting/
│   ├── persuasive-copy/       # Textos persuasivos (AIDA, PAS)
│   ├── email-marketing/       # E-mails de alta conversão
│   └── ad-copy/               # Anúncios pagos (Meta, Google, LinkedIn)
├── design/
│   ├── visual-design/         # Princípios de design visual
│   └── brand-identity/        # Identidade de marca e brand guidelines
└── ui-ux/
    ├── ui-design-system/      # Sistemas de design e componentes
    ├── ux-research/           # Pesquisa de usuário e personas
    └── landing-page-ux/       # Otimização de landing pages
```

## Skills Disponíveis

### Copywriting

| Skill              | Descrição                                                         |
|--------------------|-------------------------------------------------------------------|
| `persuasive-copy`  | Frameworks AIDA e PAS para textos persuasivos de alta conversão   |
| `email-marketing`  | Estrutura de e-mails, sequências de automação e assuntos eficazes |
| `ad-copy`          | Textos para anúncios pagos no Meta Ads, Google Ads e LinkedIn     |

### Design

| Skill              | Descrição                                                         |
|--------------------|-------------------------------------------------------------------|
| `visual-design`    | Hierarquia visual, tipografia, cores e espaçamento                |
| `brand-identity`   | Criação de identidade visual e documentação de brand guidelines   |

### UI/UX

| Skill              | Descrição                                                         |
|--------------------|-------------------------------------------------------------------|
| `ui-design-system` | Design tokens, biblioteca de componentes e atomic design          |
| `ux-research`      | Pesquisa de usuário, personas, jornadas e roteiros de entrevista  |
| `landing-page-ux`  | Estrutura, hierarquia e otimização de landing pages               |

## Como Usar

Cada skill contém um arquivo `SKILL.md` com instruções detalhadas e, opcionalmente, diretórios `templates/` e `references/` com recursos adicionais.

Para usar uma skill no Manus, instale-a através da interface e ela será automaticamente ativada para tarefas relevantes.
